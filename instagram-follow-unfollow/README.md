# Instagram Follow Unfollow
Follow and unfollow script for instagram

## How to run 
- Open list of peoples on instagram
- Open console
	- **Google Chrome**
		- ***Windows & Linux*** - Ctrl + Shift + J
		- ***Mac*** - Cmd + Option + J
- Paste code and hit Enter  

### Timeout
	 Set time of clicking button
	 1 second = 1000
