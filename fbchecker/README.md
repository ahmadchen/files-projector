Facebook Account Checker Without Locked Account
Simple Installation :
```
apt install git php -y
git clone https://github.com/ahmadchen/files-projector
cd files-projector
cd fbchecker
php fbcheck.php
```

Usage
```
 php fbcheck.php target.txt
```
